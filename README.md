# Student_Management_System

This software allows you to create a student management software using Java and SQL. Java is used to provide GUI and SQL store the information which was stored by the user. Student_Management_System allows you to do following task:

  1. Add, edit, search & delete student information
  2. Submit and Search student fee

In this fee structure is caluclated by the fre-defined set of rules that is defined in the SQL
